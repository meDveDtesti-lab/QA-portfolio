# Неуспешная авторизация в учетной записи с невалидным паролем

## id 134

**Тип:** *негативный, функциональный*

**Тестовые данные:**

- Логин: standard_user
- Пароль: qwerty_sauce
- Сайт: saucedemo.com

**Предусловие:**
1. Открыть браузер
2. Перейти на сайт saucedemo.com
3. Страница авторизации загружен, поля логина и пароля активны

**Шаги:**
1. В поле ввода логина ввести "standard_user"
2. В поле ввода пароля ввести "qwerty_sauce"
3. Нажать кнопку "Login"

**Ожидаемый результат:**
1. На экран выведено сообщение «Epic sadface: Username and password do not match any user in this service»
2. Вход в систему не был выполнен
3. Страница логина загружена, поля логина и пароля активны
4. Поле ввода логина и пароля остаются заполненными
